Section

El elemento section se usa para agrupar una sección o grupo sobre el mismo tema.

No siempre lleva un header, sirve para identificar toda la información relacionada, darle jerarquía a una página y separarla.

Aquí hay una ejemplo de Section


Puede llevar varios breaks que siguen dentro.

Article

Un elemento article se usa para identificar una sección independiente.

Se puede usar para marcar blogs, artículos, etc...

Al hacer un artículo hay que pensar si la información puede ser removida sin ninguna repercusión en el sentido.

Aside

El elemento aside contiene: sidebars, pequeñas explicaciones o información relevante. Su posición recurrente a la izquierde o derecha no es significativa con respecto a como organiza la información. Este elemento se puede anidar dentro de otras secciones. Este es un ejemplo de aside


Footer

Este elemento representa el final the una página, artículo, sección u otro. Marca el final del contenido, por lo que debe contener información no relevante y no debe divergir del documento o sección en el que se encuentra. Este es un ejemplo de footer.
