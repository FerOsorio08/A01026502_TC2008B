Background images

Es una propiedad que se puede usar para poner imagenes de fondo. Este es un ejemplo de como poner una imagen de fondo en una divisón, aun qué se puede colocar más de una.
Por defecto las imagenes se colocan en la esquina superior izquierda de un elemento.El fondo de cualquier elemento es su tamaño total desde el borde hasta el padding.


Colors Colors Colors Colors Colors Colors Colors Colors Colors Colors

Los colores en css se usan con nombres predeterminados o con valores: RGB, HEX, HSL, RGBA, HSLA. Estos también se pueden usar para establecer colores de fondo, de texto y de bordes.
Diferentes nombres de colores pueden tener el mismo valor.


Font / Font - Family

La selección de font es de suma importancia, puede hacer la diferencia al leer una página web. La letra también puede dar identidad. Es importante que siempre sea lejible ya que esta puede quitar y agregar valor al texto.
En css hay 5 font-families genéricas:

  1. Serifs
  2. Sans-serif
  3. Monospace
  4. Cursive
  5. Fantasy

Font style

El font-style es una propiedad que se usa para editar texto. Esta propiedad tiene 3 valores:

  1. normal
  2. italic
  3. oblique

Font weight

El font-weight es una propiedad que se usa para determinar el "peso" del texto. Esta propiedad tiene 2 valores:

  1. normal
  2. bold

Width, Height

En css width y height son propiedades que se usan para determinar el largo y ancho de un objeto. Estas propiedades no incluyen los marcos ni los bordes; se puede usar max-width para seleccionar el tamaño maximo de un objeto.
Estas propiedades pueden tener los siguientes valores: auto, length, % (porcentaje), initial, inherit.

Margin

Los márgenes en css se usan para determinar el margen de area de cada elemento. Tienen las siguientes propiedades: margin-top, margin-right, margin-bottom, margin-left.


Padding

La propiedad padding en css permite determinar la padding area de cualquier elemento en sus 4 lados a la vez. Tienen las siguientes propiedades: padding-top, padding-right, padding-bottom, padding-left.

Overflow

El modulo overflow en css permite que haya texto desbordado y que este se pueda escoder o verse mas abajo. En css el desbordado sucede cuando la caja de texto no puede contener el contenido completo. Existen dos tipos: ink overflow y scroll overflow, en el primero no afecta el layout y el segundo permite la opción de hacer scroll en el texto.

Este texto debe de tener las propiedades de overflow. "You can use the overflow property when you want to have better control of the layout. The overflow property specifies what happens if content overflows an element's box.""
